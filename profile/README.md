## Skills

Agent skills for Claude Code and other AI tools.

| Skill | Description |
|-------|-------------|
| [notebook](https://github.com/catcatcatai/notebook-skill) | Project notes that persist across sessions. Stop your AI from repeating the same mistakes. |
| [architect](https://github.com/catcatcatai/architect-skill) | Staged build methodology for software projects |
| [extract](https://github.com/catcatcatai/extract-skill) | Knowledge extraction from URLs and files |
| [cleanse](https://github.com/catcatcatai/cleanse-skill) | Analysis and optimization of CLAUDE.md and agent config |

Install all: `/plugin marketplace add catcatcatai/marketplace`
